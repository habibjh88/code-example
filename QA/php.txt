2. Difference between Session and Cookie?
   Ans: 
   Cookies: কুকির মাধ্যমে ইউজারকে সনাক্ত করতে লোকাল পিসিতে একটা ফাইল তৈরি করে সেখানে কিছু ডাটা রেখে দেওয়া হয়। এটা ডেস্ট্রয় হওয়ার জন্য একটা টাইম নির্ধারন করা হয়। টাইম আউট হলেই কুকুটি অটোমেটিক ডিলিট হয়ে যায়। আর এটা সাধারণত ক্লাইন্টসাইট ডাটা রাখার জন্য ব্যবহার হয়। 
   Session: সেশনও ইউজারকে সনাক্ত করার জন্য ব্যবহার হয়। মূলত ইউজার সম্পর্কে ওয়েব সারভারকে এটা জানানোর জন্য সেশন ব্যবহার হয়। যেমন ইউজারের নাম, প্রিয় রং ইত্যাদি। ডিফল্টভাবে ব্রাউজার বন্ধ না করার পর্যন্ত একটা সেশন স্থায়ী হয়। অথবা সেশন ডেস্ট্রয় করার মাধ্যমে সেশন ডিলিট করা হয়।

6. 	what are the main error type in php and how do they differ?
	Ans: There are 4 Different Error Types in PHP. 
	
	1. Parse Error (Syntax Error): The parse error occurs if there is a syntax mistake in the script. (stop execution)
	Common reason of syntax errors are: Unclosed quotes, Missing or Extra parentheses, Unclosed braces, Missing semicolon
	
	2. Fatal Error: The PHP compiler understand the PHP code but it recognizes an undeclared function. This means that function is called without the definition of function. (stop execution)
	Such as: 
	<?php
		function fun1(){
		echo "Vineet Saini";
		}
		fun2();
	?>
	
	3. Warning Error: The main reason of warning errors are including a missing file. (dosn't not stop execution)
	include ("gfg.php");
	
	4. Notice Error: It's similer to warning error. Notice that the error occurs when you try to access the undefined variable. (dosn't not stop execution)

10. how to  enable error reporting in php
	Ans: PHP has many levels of errors
	// Turn off all error reporting
	error_reporting(0);
	// variables or catch variable name misspellings ...)
	error_reporting(E_ERROR | E_WARNING | E_PARSE | E_NOTICE);

	// Report all errors except E_NOTICE
	error_reporting(E_ALL & ~E_NOTICE);

	// Report all PHP errors
	error_reporting(E_ALL);

	// Report all PHP errors
	error_reporting(-1);
	
	//php.ini file modify: display_errors = on
